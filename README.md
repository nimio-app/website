# Nimio Website

This repository contains the source code for the official Nimio website.

🌐 https://nimio.org

## About Nimio

Nimio is an open-source platform that helps people share their availability, so others know the right moment to connect.

We believe:

- Online doesn't mean available.
- Busy doesn't mean unavailable.
- Respectful communication starts before the first message.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
